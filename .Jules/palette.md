## 2024-05-18 - Added ARIA roles to Theme Switcher Menu
**Learning:** For custom animated menus (e.g. Framer Motion dropdowns), ensure the trigger element has `aria-haspopup="menu"` and `aria-expanded`, the wrapper has `role="menu"`, individual options have `role="menuitem"`, and non-interactive layout elements have `role="presentation"`. Additionally, when using `useEffect` for `localStorage` init on themes, ensure to use lazy state initialization rather than a synchronous `setState` during the effect to avoid cascading renders, and use `document.documentElement.setAttribute('class', theme)` to satisfy React mutability hooks lint rules without visual flashes.
**Action:** Always add proper `role="menu"`, `role="menuitem"`, and `role="presentation"` to non-native dropdown lists (e.g., custom Framer Motion components), and default to lazy initialization for `localStorage` state checks.
